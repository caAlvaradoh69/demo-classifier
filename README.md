# Mi Proyecto React

Proyecto React moderno creado con Vite.

## Características

- ⚛️ React 18
- ⚡ Vite para desarrollo rápido
- 🎨 CSS moderno
- 📦 Estructura lista para desarrollo

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El servidor de desarrollo estará disponible en `http://localhost:5173`

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
